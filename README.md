# Druta
Druta is a mobile-first running app with a local web auth/API companion.

## Quick Start
1. `cd druta/apps/web && bun install && bun run dev --host 0.0.0.0 --port 3000`
2. `cd druta/apps/mobile && npm install`
3. `cd druta/apps/mobile && npm run start -- --clear --port 8081`
OR `cd druta/apps/mobile && npm run web`
4. Scan the QR, open Expo Go and use the app!

Use the **Track** tab for map + live run controls, then explore **Race**, **Ranks**, and **Profile**.
